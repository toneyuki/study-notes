# Visual Studio Code ショートカット
VSCodeのショートカットで気になったものを書く

## 📖 参考サイト
- デフォルトのキーボードショートカットリファレンス
  - https://code.visualstudio.com/docs/reference/default-keybindings

## ショートカット
- キーボードショートカットを開く（Ctrl+K Ctrl+S）

- 検索を開く（Ctrl+F）
  - 正規表現検索の切り替え（Alt+R）

- 置換を開く（Ctrl+H）
  - （全て置換するはCtrl+Alt+Enter）

- 選択行を切り取り（Ctrl+X）

- 選択行を削除（Ctrl+Shift+K）

- 選択行を下行にコピー（Shift+Alt+↓）

- 下方向への矩形選択（Ctrl+Shift+Alt+↓）
  - （Ctrl+Shift+Altを押したまま、矢印キーで選択する。エスケープで解除する。矩形の読み方は「くけい」）

- プレビューを横に表示（Ctrl+K → V）
  - （マークダウン記法の.mdファイルの場合、最新のプレビューが横に表示されるのでめちゃくちゃ便利）

- アクティブファイルの絶対パスをコピー（Shift+Alt+C）

- アクティブファイルの相対パスをコピー（Ctrl+K → Ctrl+Shift+C）

- ターミナルを開く（Ctrl+@）

- ターミナルの分割（Ctrl+Shift+5）

- 新しい外部ターミナル（WindowsTerminal）を開く（Ctrl+Shift+C）

- （Ctrl + Alt + V）
